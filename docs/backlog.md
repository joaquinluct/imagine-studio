# Backlog (baúl)

> **Referencia**: Ver [`docs/MAIN.md`](MAIN.md) para los pilares fundamentales del proyecto.

Este fichero contendrá, de forma persistente, todos los ítems que NO están en el sprint activo y que quedan para futuras iteraciones.

Por ahora está vacío y se usará como depósito para historias/tareas descartadas temporalmente o pendientes de priorización.

---

## Formato

Lista de elementos con:
- **ID**: Identificador único
- **Prioridad**: Baja/Media/Alta
- **Descripción**: Breve descripción del ítem
- **Referencia**: Enlace a historia/tarea en `docs/sprint_histories.md` o `docs/sprint_tasks.md` si procede
